1. Vim的基本指令

在新接触Vim的时候，诸多指令令人难以入手，最有效的办法就是多加练习和使用，这次的Markdown文件就是使用VIM编写，记录下来以方便查找，如果以后使用到了其他的指令再更新。

1、文件定位

指令 功能
行号 + G 定位到具体行
G 定位到文件最后一行
gg 定位到文件第一行
CTRL + G 显示当前打开的文件名和所在行
$ 行的末尾
0 行的开头
w 移到下一个词的词首
W 移到下一个词的词首
b 移到上一个词的词首:
B 移到上一个词的词首
e 移到下一个词的词尾
E 移到下一个词的词尾
ge 移到上一个词的词尾
gE 移到上一个词的词尾
5w 向前移动五个单词，其他的类似

注意w与W，b与B，e与E，ge与gE的区别以如下案例阐述：

例如对于字符串root/usr bin，光标在r处，w移动到/处，W移动到b。

2、滚屏

指令 功能
ctrl-u 文本向下滚到半屏。光标不随文本移动。相当于文本的窗口上移半屏
ctrl-d 窗口向下移动半屏，相当于文本向上滚动了半屏
ctrl-e 向上滚动一行
ctrl-y 向下滚动一行
ctrl-f 向前滚动一屏(forward)
ctrl-b 向后滚动一屏(backward)
zt 把当前行置于屏幕顶端
zb 把当前行置于屏幕底
zz 把当前行置于屏幕中央

3、浏览文档

指令 功能
( 跳到上一句
) 跳到下一句
{ 跳到上一段
} 跳到下一段
[[ 调到上一部分
]] 跳到下一部分
[] 跳到上一部分末尾
][ 调到上一部分开头

4、插入模式

指令 功能
i 在光标所在字符 进入insert模式
a 在光标的下一个字符 进入insert模式
A 光标所在行末插入一行 进入insert模式
o 在光标下一行新建一行 进入insert模式
O 在光标上一行新建一行 进入insert模式

5、特殊插入

指令 功能
:r [filename] 将该file里的内容插入到光标下方
:r ![command] 将command(Linux指令)的结果插入到光标下方

6、可视化模式(用来选中文本)

指令 功能
v 在光标所在字符 进入visual模式
V 在光标所在的行 进入visual模式
Ctrl + v 在光标处进入块选中模式

7、复制黏贴剪

指令 功能
y 点击v进入可视模式，复制选中文本
p 在当前行之后粘贴寄存器当中的内容
P 在当前行之前粘贴寄存器当中的内容
dd 既是删除行，也是剪切，使用p粘贴

8、删除

指令 功能
X 删除光标所在位置的字符
dw 删除光标所在位置的单词
d0 删除光标至行首
d$ 删除光标至行末
dd 删除整行
3dd 删除光标后三整行
dgg 删至文件开头
dG 删至文件结尾
d) 删至句末

9、查找和替换

指令 功能
/ + 查找的字符（进入查找模式） 从前向后查找文件中所有的该字符
下一个对象：n 移动到下一个该字符处
上一个对象：N 移动到上一个该字符处
? + 查找的字符（进入查找模式） 从后向前查找文件中所有的该字符
r{text} 将该光标的字符替换成 text
R 进入覆写模式
%s /original/replacement 将第一个original词替换成replacement
%s /original/replacement/g 将所有的original词替换成replacement
%s /original/replacement/gc 将所有的original词替换成replacement，替换前先询问

10、设置书签

指令 功能
m {a~zA~Z} 当前光标处设置一个名为…的书签
:marks 列出所有书签
{a~zA~Z} 跳转至…书签

11、编辑

指令 功能
~ 变化为大写
J 合并下一行到上一行
[N]>> 将一行或N行右移一个tab
[N]<< 将一行或N行左移一个tab

12、撤销与重做

指令 功能
u 撤销最后执行的命令（最后一次的命令）
U 撤销对整行的修改（最后一次的批量命令）
Ctrl + r 重做最后撤销的操作
. 重做上一个指令

13、保存退出

指令 功能
:q 退出
:w 保存
:wq 保存退出
:q! 强制退出
:w new_name 另存为new_name

14、配对字符查找

在一行中的任何一个 (、[ 或 { 处，输入 % 可以查找配对的括号 )、]、}

15、vim多窗口模式

指令 功能
:vs newfile 水平打开一个新窗口
:sp newfile 竖直打开一个新窗口
:split 水平分割出一个窗口(同一文件)
:vsplit 竖直分割出一个窗口(同一文件)
Ctrl + w 换窗口
Ctrl + w + h 向左移动窗口
Ctrl + w + j 向下移动窗口
Ctrl + w + l 向上移动窗口
Ctrl + w + k 向右移动窗口
Ctrl + w + w 这个命令会在所有窗口中循环移动
Ctrl + w + t 移动到最左上角的窗口
Ctrl + w + b 移动到最左上角的窗口
Ctrl + w + p 移动到前一个访问的窗口

16、其他

指令 功能
:! + xxx外部命令 如 :!ls，在shell中执行ls
